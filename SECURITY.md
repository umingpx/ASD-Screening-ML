# Security and Privacy Policy

## Research Integrity
This repository contains a screening prototype developed for scientific research and educational purposes. As a student project, this tool is not intended for clinical use in a production environment.

## Data Privacy (PDPA Compliance)
User privacy is a priority for this project. The current Streamlit deployment is designed to be "stateless." This means:
*   No personal identifiable information (PII) is collected.
*   The data entered into the screening questionnaire is processed in real-time to generate a result and is not stored in any database.
*   The model weights (`.pkl`) are used for inference only and do not contain any raw patient data from the training set.
